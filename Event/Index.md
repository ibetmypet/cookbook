# Event

Här samlas planerade mat- och dryckesaktiviteter. Ett event kan länka till recept, menyer, inköpslistor, provningsmaterial och anteckningar efteråt.

## Eventtyper

- [[Ölprovning/Index|Ölprovning]]
- Middagar
- Grillkvällar
- Temakvällar
- Matprovningar
- Högtider

## Arbetsflöde

1. Skapa eventet från [[../Mallar/Eventmall|Eventmallen]].
2. Ange deltagare, datum, upplägg och budget.
3. Länka till recept, menyer och inköpslista.
4. Skapa en tydlig tidslinje för förberedelser och servering.
5. Dokumentera resultat och förbättringar efter eventet.
